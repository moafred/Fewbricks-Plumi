# Plumi : Objectifs Pédagogiques CE1 (2026)

Ce document définit les standards pédagogiques "Attendus de fin d'année" pour le cycle CE1, conformément à la réforme **"Choc des Savoirs"** (Bulletin Officiel n° 41 du 31 oct 2024).

---

## 1. Grammaire [PEDA]

L'objectif est de passer d'une intuition du langage à une identification structurée des composants de la langue.

### Identifier la Phrase
- **Capacité** : Reconnaître une phrase (majuscule, point, sens).
- **Types & Formes** : Distinguer phrase déclarative et interrogative. Maîtriser la forme négative (*ne... pas*).
- **Groupes** : Identifier le **Groupe Sujet** (GS) et le **Verbe** dans des phrases simples.

### Classes de Mots
| Classe | Attendu CE1 |
| :--- | :--- |
| **Nom** | Distinguer nom propre et nom commun. |
| **Déterminant** | Identifier les articles (le, la, un, une...) et possessifs simples (mon, ton, son...). |
| **Verbe** | Identifier l'action ou l'état dans la phrase (mots qui se conjuguent). |
| **Pronom** | Connaître les pronoms personnels sujets (je, tu, il/elle/on, nous, vous, ils/elles). |
| **Adjectif** | Identifier le mot qui qualifie le nom dans le Groupe Nominal (GN). |

### Accords & Relations
- **Accord sujet-verbe** : Appliquer la marque de personne au verbe selon le sujet.
- **Accord dans le GN** : Maîtriser l'accord en genre (masculin/féminin) et en nombre (singulier/pluriel) entre le nom, le déterminant et l'adjectif.

---

## 2. Conjugaison [PEDA]

L'élève doit automatiser les flexions verbales pour les temps fondamentaux.

### Les Temps de Référence
1. **Présent** : Le temps pilier du CE1.
2. **Futur** : Compréhension de l'action à venir.
3. **Imparfait** : Première approche du récit passé.
4. **Passé Composé** : Introduction via le participe passé pour les verbes fréquents.

### Verbes à Maîtriser (Présent, Futur, Imparfait)
- **Auxiliaires** : *Être* et *Avoir*.
- **Verbes du 1er groupe** : Tous les verbes se terminant en -ER (ex: *jouer, manger, chanter*).
- **Verbes irréguliers fréquents** : *Aller, Faire, Dire*.

### Régularités de Personne
- Connaissance des terminaisons régulières (ex: -e, -es, -e, -ons, -ez, -ent pour le présent).
- Distinction auditive/écrite (ex: chantes / chantent).
